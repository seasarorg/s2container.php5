REQUIREMENT

S2Container.PHP5 requires PHP5 to run. The operation test is checked in PHP-5.0.5, PHP-5.0.4.

SETUP

  • Download S2Container.PHP5.version.zip, and Unzip it!
s2container.php5/
  +-- docs/                ----- document directory
  +-- src/                 ----- source directory
  +-- test/                ----- test directory
  +-- SEASAR-LICENSE.TXT   ----- Seasar license file
  +-- s2container.inc.php  ----- S2Container.PHP5 initialize file
  • Require initialize file.
<?php
require_once('/path/to/s2container.php5/s2container.inc.php');
?>
  • You can find a sample in src/examples.

AUTOLOAD

S2Container.PHP5 uses __autoload() function to require a class definition. __autoload() function is defined in s2container.inc.php.

<?php
・・・
function __autoload($class=null){
    if(S2ContainerClassLoader::load($class)){return;}
}
・・・
?>
If you have already defined __autoload() function in your application or in some other framework, just comment lines out. 
<?php
・・・
/*
function __autoload($class=null){
    if(S2ContainerClassLoader::load($class)){return;}
}
*/
・・・
?>
Then, put this line into defined __autoload() function. 
if(S2ContainerClassLoader::load($class)){return;}
*** s2container.inc.php must be required before definition of __autoload() function.
