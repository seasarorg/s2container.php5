PHP scripting in dicon file

PHP scripting is available in <property> tag, <component> tag, <initMethod> tag, <destroyMethod> tag, <arg> tag, <meta>tag and <aspect> tag. There are examples located in s2container.php5/src/examples/expression .

arg, property, aspect, meta tag

In arg, property, aspect, meta tag, you can write PHP scripting which returns values or references to other component.

<?xml version="1.0" encoding="Shift_JIS"?>
<!DOCTYPE components PUBLIC "-//SEASAR2.1//DTD S2Container//EN"
"http://www.seasar.org/dtd/components21.dtd">
<components>
    <component name="sample1" class="ArgExp">
        <arg>10</arg>
        <property name="message">
            file_get_contents(EXAMPLE_DIR . '/dicon/expression/ArgSample.txt')
        </property>
    </component>
    <component name="sample2" class="ArgExp">
        <arg>2+3</arg>
        <property name="message">
            $msg1 = "Hello ";
            $msg2 = "World";
            return $msg1 . $msg2;
        </property>
    </component>
</components>

component tag

Writing PHP scripting which returns an object.

<?xml version="1.0" encoding="Shift_JIS"?>
<!DOCTYPE components PUBLIC "-//SEASAR2.1//DTD S2Container//EN"
"http://www.seasar.org/dtd/components21.dtd">
<components>
    <component name="hello">
       new ComponentExp("Hello")
        <property name="messageB">"World"</property>
    </component>
    <component name="hello2" class="ComponentExp">
        hello3
    </component>
    <component name="hello3" class="ComponentExp"/>
</components>
If class attribute of component tag is set, the object returnd by PHP scripting is compared with class attribute value by ReflectionClass#isSubclassOf.

initMethod, destroyMethod tag

You are able to write PHP scripting in initMethod and destroyMethod tag. Variable [ $component ] references self component.

<?xml version="1.0" encoding="Shift_JIS"?>
<!DOCTYPE components PUBLIC "-//SEASAR2.1//DTD S2Container//EN"
"http://www.seasar.org/dtd/components21.dtd">
<components>
    <component name="method" class="MethodExp">
        <initMethod>
            $component->init(10)
        </initMethod>
    </component>
</components>
