セットアップ

s2container.php5.version.zipを取得し解凍して下さい。以下のディレクトリが展開されます。 

s2container.php5/
  +-- docs/                ----- ドキュメント ディレクトリ
  +-- src/                 ----- ソース ディレクトリ
  +-- test/                ----- テスト ディレクトリ
  +-- .project             ----- Eclipseのプロジェクトファイル
  +-- SEASAR-LICENSE.TXT   ----- Seasarライセンス
  +-- s2container.php5.ini ----- S2Container.PHP5 設定ファイル

s2container.php5.ini を読み込んで下さい。S2Container.PHP5が使用可能となります。 

<?php
require_once('/path/to/s2container.php5.ini');
?>

クラス定義の AutoLoad について

S2Container.PHP5 では、__autoload関数を用いてクラス定義ファイルを読み込みます。
__autoload関数は、s2container.php5.ini ファイル内で定義されています。 

<?php
・・・
function __autoload($class=null){
    if(S2ClassLoader::load($class)){return;}
}
・・・
?>
開発中のアプリケーションや使用しているフレームワークで、既に__autoload関数が定義されている場合は、s2container.php5.ini 内の __autoload関数をコメントアウトして下さい。 
<?php
・・・
/*
function __autoload($class=null){
    if(S2ClassLoader::load($class)){return;}
}
*/
・・・
?>
s2container.php5.ini 内の __autoload関数をコメントアウト後、以下の1行をご使用の__autoload関数の先頭に挿入して下さい。 
if(S2ClassLoader::load($class)){return;}
***注） __autoload関数定義が読み込まれる前に s2container.php5.ini ファイルの require を行って下さい。
***注） S2ClassLoader::load メソッドの引数($class)は、__autoload関数の引数を渡して下さい。
